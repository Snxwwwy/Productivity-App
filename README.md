# Productivity App

A clean and simple **task management app** designed to help users organize their daily tasks, track priorities, and boost productivity. Built using **HTML, CSS, and JavaScript**.

---

 Features

- Add, delete, and mark tasks as complete  
- Task categories / tags (Work, Personal, Shopping, etc.)  
- Priority labels (High, Medium, Low)  
- Subtasks / Checklists for breaking down tasks  
- Due dates and calendar view  
- Drag-and-drop sorting of tasks  
- Dark mode and responsive design for all devices  
- Offline mode (tasks stored in browser)  

> Note: Advanced features like collaboration, AI reminders, and third-party integrations are planned for future updates.
